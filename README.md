## コミットメッセージ

| name   | effect                 |
| ------ | ---------------------- |
| add    | 新しいファイルやコード |
| fix    | バグの修正             |
| delete | ファイルやコードの削除 |
| change | 仕様変更               |

ex.(add: ユーザーネーム登録機能)
